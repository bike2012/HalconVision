# 第一部分：圆度误差的测量及评定方法
	定义：圆度误差的概念来自于工程实际 , 按照国际标准ISO1101 的规定, 圆度误差为包容实际圆轮廓的半径差为最小的两同心圆的半径差 
	测量方法：
	1.传统测量法：比较检验法、特征参数测量法、坐标测量法（是利用光学分度头、分度台 、坐标测量机等测量装置, 采用布点测量被测轮廓
	  的极坐标或平面直角坐标的一种测量方法）
	2.新式测量法：计算机视觉检测（基于工件正截面摄像的圆度误差视觉检测系统包括五个部分 :获取图像 , 一般是通过 CCD 摄取圆形截面
	  的正投影图像 ;图像预处理 ,对摄取的图像滤波去噪,图像增强处理 ;圆轮廓提取 ,对预处理后的图像 ,提取图像边缘或跟踪圆轮廓得到图
	  像要素;圆度误差计算,对获取的圆轮廓像素进行误差评定;最后,显示结果。基于工件轴截面摄像的圆度误差视觉检测方法是通过 CCD 摄取
	  圆轴侧面正投影图像, 计算侧面图像各截面的直径 ,然后将圆轴旋转一定角度, 再摄取其侧面正投影图像并计算其各截面直径, 后按各直径
	  值评定圆度误差）、采用误差分离技术的测量方法、圆度误差的在线检测、光学原理衍生的圆度测量方法
	圆度误差评定方法：
	1.最小二乘法(LSM)
	2.最小区域法(MZC)
	3.最大内接圆法(MIC)与最小外接圆法(MCC)
	4.基于区域搜索的圆度误差评定方法
	5.智能优化算法评定圆度误差
	6.以小波分析为基础的评定算法
	7.以支持向量机理论发展的评定算法
	8.以支持向量机理论发展的评定算法
	9.全局收敛高斯 —牛顿法
	10.单纯形法
